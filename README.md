# ☁️ Weather-webApplication

This is a simple and responsive web application that allows users to instantly view the current weather conditions for any specified city around the world.

The application fetches real-time meteorological data using a third-party API and presents it in a clean, user-friendly interface.

## Live Demo

You can interact with the live application here:
https://amrath049.github.io/Weather-webApplication/

## Features

* **Real-time Data:** Fetches up-to-date weather information using an external API.
* **Location Search:** Users can input any city name to retrieve weather details.
* **Key Metrics Displayed:** Shows essential information including:
    * Temperature (in Celsius)
    * Weather Description (e.g., 'Few Clouds')
    * Humidity Percentage
    * Wind Speed
* **Intuitive Design:** Clean, modern interface suitable for all users.

##  Technology Stack

This project was built using core web development languages and a third-party API:

| Category | Technology | Purpose |
| :--- | :--- | :--- |
| **Structure** | HTML5 | Provides the basic layout and content elements. |
| **Styling** | CSS3 | Ensures responsiveness and a modern aesthetic. |
| **Functionality** | JavaScript (Vanilla JS) | Handles user input, API calls, and dynamic content updates. |
| **Data Source** | OpenWeatherMap API | Used to fetch accurate and current weather data. |
